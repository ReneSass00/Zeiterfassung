# Time Tracking Application (Full-stack .NET)

This project is a dedicated time-tracking web application developed as a private project to master modern web architectures. The primary focus was on implementing a clean architecture and leveraging Blazor for a highly reactive user interface.

##  Features
* **Time Tracking:** Efficient logging of working hours, breaks, and specific projects.
* **Dashboard:** Start time-tracking, overview about worked hours, project hours distribution and last activities 
* **Modern UI:** Responsive design using Blazor components powered by the MudBlazor library.
* **Data Persistence:** Robust database integration via Entity Framework Core.
* **Reporting:** Automated PDF generation and reporting using QuestPDF.
* **Best Practices:** Implementation of Dependency Injection (DI) and asynchronous programming patterns (async/await).

## Tech Stack
* **Frontend:** Blazor (C#)
* **Backend:** .NET 9 / ASP.NET Core
* **Persistence:** Entity Framework Core (SQL Server)
* **UI Library:** MudBlazor
* **Reporting:** QuestPDF
* **Tools:** Visual Studio, Git
