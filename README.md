# Zeiterfassungs-Anwendung (Fullstack .NET)

Dieses Projekt ist eine Web-Anwendung zur Zeiterfassung, die ich privat entwickelt habe. Der Fokus lag dabei auf einer sauberen Architektur und der Nutzung von Blazor für eine reaktive Benutzeroberfläche.

##  Features
* Zeiterfassung: Erfassung von Arbeitszeiten, Pausen und Projekten.
* Modernes UI: Responsive Design mit Blazor-Komponenten unter Verwendung von MudBlazor.
* Datenhaltung: Anbindung an eine Datenbank via Entity Framework Core.
* Reporting mit QuestPDF
* Best Practices: Nutzung von Dependency Injection und asynchroner Programmierung.

## Tech Stack
* Frontend: Blazor (C#)
* Backend: .NET 9 / ASP.NET Core
* Persistence: Entity Framework Core (SQL Server)
* Tools: Visual Studio, Git
